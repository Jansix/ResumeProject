  • 課程系統

  • 此專案為MERN專案，以MongoDB資料庫、Express伺服器框架、React前端框架、NodeJS後端伺服器搭建而成。功能包含用戶註冊、登入、新增/刪除課程、選課、搜尋等。

    - 前端技術： 使用Bootstrap 框架建立響應式使用者介面，透過Axios發送 AJAX請求與後端通信，使用 React 構建動態的前端應用，根據不同的用戶身分處理可看見的畫面，並使用localStorage儲存用戶狀態以提供持久性登入體驗。

    - 後端技術：以REST架構設計API，透過cors解決跨域問題，joi用以驗證資料是否符合格式，另外後端除CRUD外，主要側重在用戶帳號相關的驗證機制上，系統採用bcrypt對用戶密碼進行雜湊加密，以jsonwebtoken方式實現無狀態身分驗證，提升資料安全性。

  • 個人網頁(本站)

  • 此網頁使用原生 JavaScript、CSS 和 HTML 架設，除了基本頁面和樣式呈現外，還包含以下現代網頁常見功能：

    - 完全響應式設計(RWD)： 網頁能根據不同設備大小和解析度自動適應布局和樣式。
    - 摺疊選單(hamburger)：提供可折疊的菜單，增加導航的便利性。
    - 明暗模式切換(Dark Mode)：設計明暗兩種主題模式供用戶選擇，以適應不同的環境和用戶喜好。
    - 回到頁首按鈕(BackToTop)：放置便捷按鈕，使用戶能夠快速回到頁面頂部。
    - 段落錨點(anchor)：設置錨點自動跳轉到指定段落，提升用戶體驗。
    - 動畫特效：單純使用css完成網頁泡泡動畫特效，進一步強化CSS熟練度。
